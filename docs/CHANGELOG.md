### Added

-   Daily journal entry creation with YAML frontmatter
-   Automatic directory structure (`journal/yyyy/mm/yyyy-mm-dd.md`)
-   Visual indicators for days with entries
-   Weekend highlighting in calendar
-   Last opened day tracking
-   Month navigation (previous/next)
